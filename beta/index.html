So installieren Sie die CEDRUMO-Testversion

Bitte öffnen Sie den TestFlight-Link direkt auf Ihrem iPhone.

  1. Installieren Sie zuerst Apples kostenlose App TestFlight aus dem App Store.
    TestFlight im App Store öffnen
    Nach der Installation kehren Sie zu dieser Seite zurück und fahren mit Schritt 2 fort.
  2. Tippen Sie anschließend unten auf den Einladungslink.
    CEDRUMO-Einladung öffnen
  3. Scrollen Sie auf der Apple-Seite nach unten, bis „Cigar Collection by CEDRUMO“ erscheint.
  4. Tippen Sie auf „In TestFlight anzeigen“.
  5. Installieren Sie die CEDRUMO-Testversion.
Wichtig: Es ist kein Einladungscode erforderlich.
Falls eine Fehlermeldung erscheint: Meldet Safari, dass die Seite nicht geöffnet werden kann, weil die Adresse ungültig ist, ist TestFlight noch nicht auf dem iPhone installiert. Bitte installieren Sie zuerst TestFlight und öffnen Sie anschließend den Einladungslink erneut.

How to install the CEDRUMO test version

It only takes a few minutes, and you'll be among the first to experience CEDRUMO. Please open the TestFlight link directly on your iPhone.

  1. First, install Apple’s free TestFlight app from the App Store.
    Open TestFlight in the App Store
    After installing, come back to this page and continue with step 2.
  2. Then tap the invitation button below.
    Open the CEDRUMO invitation
  3. Scroll down on the Apple page until you see “Cigar Collection by CEDRUMO.”
  4. Tap “View in TestFlight.”
  5. Install the CEDRUMO test version.
Important: No invitation code is required.
If you see an error: If Safari says the page cannot be opened because the address is invalid, TestFlight is not yet installed on the iPhone. Please install TestFlight first and then open the invitation link again.
